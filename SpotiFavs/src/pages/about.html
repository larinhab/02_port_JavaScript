Sobre o Projeto

A aplicação consiste em uma buscar por um perfil na Plataforma do Spotify, quando a busca é concluida a tabela mostra o artista, a música, e o perfil pesquisado. Você pode pesquisar por outros perfis. Também pode remover caso um perfil não seja mais do seu interesse. Assim que concluir pode voltar à pagina inicial e recomeçar. A barra de navegação é fixa e nela você pode interagir com as páginas.


Esta aplicação utiliza as tecnologias: HTML, CSS, JAVASCRIPT, NODE JS E API DO SPOTIFY.


Translate

The application consists of searching for a profile on the Spotify Platform, when the search is completed the table shows the artist, the song, and the searched profile. You can search for other profiles. You can also remove if a profile is no longer of interest to you. Once finished, you can return to the home page and start over. The navigation bar is fixed and you can interact with the pages there.


This application uses the following technologies: HTML, CSS, JAVASCRIPT, NODE JS AND SPOTIFY API.
